# 决策记录索引（ADR）

每条记录用同一份模板：**背景 / 选项 / 决定 / 理由 / 与 AI 讨论摘要 / 状态**。

新增决策：复制最近一份 .md，编号 +1，标题用中文短语。

---

## 状态图例

- ✅ **生效中** — 当前在按这个决策做事
- 🔄 **已演进** — 被后续 ADR 修订或替换
- ❌ **已废弃** — 决策被推翻，保留作历史

## 模板

```markdown
# NNNN — 标题

- **日期**：YYYY-MM-DD
- **状态**：✅ 生效中 / 🔄 已演进（→ NNNN） / ❌ 已废弃
- **影响范围**：vocab-review / listening-player / shared / 全局

## 背景

为什么会出现这个决策？此前的状态是什么？什么事件触发了讨论？

## 选项

考虑过哪些方案，每个的优劣。

## 决定

最终选了哪个，怎么落地。

## 理由

为什么是这个，不是其他的。把"用户视角"和"工程视角"分开写。

## 与 AI 讨论摘要

讨论中 AI 提出但被否决的点 / 提醒到的盲区 / 关键的 push back。

## 状态

后续如果被修订，在这里挂链接到新 ADR。
```
